# My Dream Destination

A simple static landing page showcasing a dream travel destination: Japan.

## Project Structure

- `index.html` - Main homepage with destination overview, attractions, travel plan, and useful links.
- `assets/css/style.css` - Basic page styling and background color.
- `assets/image/` - Image folder used by the homepage.
- `public/contact.html` - Contact page with email and phone details.

## Features

- Destination summary for Japan
- Top attractions list
- Suggested travel activities
- Budget table for Tokyo, Kyoto, and Osaka
- External travel resource link
- Contact page link

## How to view

1. Open `index.html` in a web browser.
2. Click the `contact us` link to visit the contact page.
